Sobre a API

API utilizada: cataas
cat as a service

A API escolhida gera uma imagem aleatória de um gato e suas tags associadas, se existente.

Quando é feita a request, é gerado um Json com dados sobre a imagem selecionada, como: ID da imagem, tags, data de upload e tamanho original.

No arquivo TS do componente, esse Json é tratado e somente as informações importantes são guardadas, nesse caso o ID e as tags associadas a imagem gerada.
